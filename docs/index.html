Machine Intelligence

Powered by Redis Stack vector search and CLIP embeddings, Photarium understands your images semantically — not just by filename.

CLIP Embeddings

512-dimensional semantic vectors capture meaning, mood, and concept — a lonely chair neighbors a portrait of solitude.

Color Histograms

64-dimensional color vectors for palette-based search. Find all your sunset oranges or ocean blues.

Semantic Search

Type natural language like "vintage car at sunset" and find conceptually matching images, no tags required.

Concept Radar

Visualize how the machine perceives your image across 10 aesthetic axes: warm↔cold, minimal↔complex, nostalgic↔futuristic.

Semantic Clustering

See neighbors and antipodes to explore your collection's semantic topology.

Click-to-Search Radar

Click anywhere on the concept radar to search for images matching those semantic coordinates. Interactive exploration.

Machine Haiku

Generate contemplative 5-7-5 haiku from CLIP embeddings. The machine's poetic interpretation of your image's essence.

Embedding Ops

Batch embeddings, backfills, and status visibility keep large libraries searchable.

Redis Stack

High-performance vector similarity search via RediSearch. Sub-millisecond KNN queries across your entire library.

Color Search

Search by hex color like #3B82F6 to find images with matching palettes. Perfect for brand consistency.

Precision Control

Manage metadata, edit tags, and assign image variants with granular precision. Keep your library organized by grouping related assets (like thumbnails or social crops) under a single master image.

Organized Gallery Management

Browse thousands of assets with smart filtering, sticky controls, and instant quick actions. Find exactly what you need in seconds.

Get Started in Minutes

1

Create Cloudflare Account

Sign up free, then enable Cloudflare Images.

2

Clone & Install

git clone https://github.com/bleeckerj/nfl-photarium.git and npm install.

3

Add Credentials

Copy .env.example to .env.local and add your Cloudflare details.

4

Run Locally

npm run dev and open http://localhost:3000.

5

Enable Vector Search (Optional)

Run Redis Stack and generate embeddings for semantic search and similarity.

6

Deploy Anywhere

Deploy to Vercel, Railway, Render, or any Node.js host.

7

Start Managing

Upload images, organize with folders and tags, copy CDN URLs.
